Select Units

There is another way to control units.

Dota 2 has these fantastic features:
Select Hero
Select All Other Units
Select All Units
Unified Orders

Select Hero is self-explanatory.
The best thing about this bind is that you don't need it.

Just assign your hero to 1Control Group 1 instead.
This gives you the same result, but provides additional flexibility of a control group.

Example:
Naga Siren players usually bind the main hero and one illusion together.

Select All Other Units selects all units under your control except your hero.

Select All Units selects all units under your control.
In other words, Select Hero + Select All Other Units.

Unified Orders is a modifier key, Ctrl by default
Unified Orders allows you to command all units at once without selecting them.

Fantastic bind, comes very handy when you summoned a bunch of stuff that you haven't yet bothered to hotkey properly.

There are 4 ways to approach these keybinds:

Option A - Bind as ALT Control Groups
Option B - Bind as top layer in place of Control Groups
Option C - Bind as a 1F1` triangle
Option D - Bind to some leftover keys

There are advantages and disadvantages to each.
If you're this deep, I believe I can elaborate.

Let's consider Option A:

  • Select Hero 1   (set to Control Group 1)
  • Select All Other Units Alt   +   1
  • Select All Units Alt   +   2

This makes sense.

But the problem is that we already use these buttons for our Shop actions.

The second problem is Alt + key might be not very fast to use as our 'Select' keys.

Not ideal.

Let's consider Option B:

  • Select Hero 1   (set to Control Group 1)
  • Select All Other Units 2
  • Select All Units 3

The main advantage here is that it's the fastest way to use these actions.

The downside is that you lose access to control groups 2 and 3.

So this is worse when you want to control multiple individual units in multiple places.
Admittedly, that's pretty rare unless you're really pushing it.

And to be fair, you can still bind individual units to 4 5 and beyond.
Considering that you get instant access to Select All Other Units and Select All Units it might be worth it.


Another somewhat notable downside is that you lose continuity with other games in which 123 are usually the control groups.

Let's consider Option C:

If you're willing to sacrifice F1 and ` you might end up with something like this

  • Select Hero 1   (set to Control Group 1)
  • Select All Other Units `   or   F1
  • Select All Units F1   or   `

This "triangle" variation works because ` and F1 are close to 1, so all your "Select" keys are super close to each other in a way that makes sense.

However.

1. You'd have to find new keys for Select Courier and Upgrade Talent.

2. Juggling 1F1` is pretty awkward, at least I find it so.

3. F1` force your hard too far away from the rest of the keys that you want to have immediate access to.

Personally - no.

Option D

  • Select All Other Units WheelDown
  • Select All Units WheelDown   +   Alt

Now, these are weird binds.
But this is what I'm currently using.
Sometimes.

I manage most of my unit control using Control Groups anyway.
So I just have these bound to something that's still available.

Fun Fact!

Naga Siren has illusions, but Terrorblade has "images".

The difference?

You cannot assign individual images to different control groups.

When you assign an image to a control group - that control group always recalls all images.

Not sure if intended.
