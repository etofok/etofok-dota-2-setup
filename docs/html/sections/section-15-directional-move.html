Directional Move

Sometimes you want to turn your hero towards a direction exactly.

That's why Directional Move is used all the time on heroes like
Shadow Fiend,
Slark,
Ursa,
Earth Spirit.

Directional Move requires some practice, but it's a better way to line up your spells.
There are 2 ways to bind it.

CapsLock is a great key for Directional Move, because we can use it with the pinky finger.

This allows us to use all the abilities and items as usual.
Zero problems.

  • Directional Move CapsLock

With the August 19, 2025 update Valve added this:

However, I do not recommend using this.

Because, while yes it's more convenient for Directional Move it really makes everything else much worse.

Think about it, we use Alt for everything:
- Self-casting abilities,
- Self-casting items,
- Toggling abilities,
- Toggling items (wards),
- All things shop and courier,
- Pinging on things,
- Highlighting spawnboxes, timers and tower ranges

So this setting gets in the way. Really.
I playtested it a lot.

Just use a "normal" key like CapsLock.

Fun Fact!

This setting is very old, but you needed to know this console command:

  • bind "ALT" "+dota_directional_move"
  • cl_dota_alt_unit_movetodirection true
