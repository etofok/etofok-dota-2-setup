Abilities No Target

One last thing about abilities.

If a No Target Ability is bound to Q, then AltQ will not activate it.
The input is 'consumed' with no effect.

And because we're going to use ALT all the time - we don't want that to happen.

Bind these abilities to Normal Cast and Quick Cast

Normal Cast - Q
Quick Cast - AltQ

All we're doing here is making sure that both Q and AltQ point to Mirror Image so that it triggers on Q and AltQ.

Same with Song of the Siren with R and AltR respectively.

Same with all the other No Target Abilities.
