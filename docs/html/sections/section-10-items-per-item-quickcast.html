Per-Item Quickcast

People usually miss this option entirely.
Valve made it more visible in the 7.39 patch.
(May 23rd, 2025)

Very few people even aware of this option, but it's the best.

This menu allows you to set Normal Cast per item, overriding Quick Cast slot binds.

Uncheck the following:

Observer and Sentry Wards

Meteor Hammer

Gleipnir

Regardless of the inventory slot, these items will use Normal Cast.

This is critical because you want to see the Area Of Effect visual indicator before use.
