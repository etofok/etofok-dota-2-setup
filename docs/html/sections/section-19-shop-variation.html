Shop Variation

Believe it or not, I actually used ALTMouseWheelUp for Open Shop for about a year.

I liked it.

Despite how strange it sounds it plays very well, because you 'scroll up to see a different layer'.
I promise it's not bad.

But the main benefit is that it allows you to shift the whole shop thing 1 key to the left:

  • Open Shop Alt   +   MouseWheelUp
  • ← Quickbuy Alt   +   1
  • ← Courier Deliver Alt   +   2
  • ← Courier Burst Alt   +   3
  • ← Buy Sticky Alt   +   4

This is technically a more optimized layout.
Although less intuitive.

You also lose continuity with other games that might not allow you to bind AltMouseWheelUp as your shop / inventory key.

This variation is unarguably 1 key more compact and feels slightly faster.
However, I use the "simple" version now.
