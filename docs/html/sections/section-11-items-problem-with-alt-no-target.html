Items Problem: Alt + No Target

Holding down ALT blocks user input for No Target items (and abilities).
The input is 'consumed'.

This happens because you can't "target" a No Target item / ability, so it gets ignored.

Here's an example.
Let's say, we have these items:

Glimmer Cape,
Gale Guard,
Eul's Scepter of Divinity
Magic Stick

To use these items on ourselves we have to juggle the ALT key:

1. To Glimmer Cape myself, I have to use ALT
2. To Gale Guard myself (it can't be targeted), I have to release ALT
3. To Eul's Scepter of Divinity myself, I have to press ALT again
4. To use Magic Stick (it can't be targeted) I have to release ALT

I don't see the reason why it should be this way.
Hopefully it'll get changed.
