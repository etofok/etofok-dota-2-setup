Items Problem: Alt + No Target

Holding down ALT blocks user input for No Target items (and abilities).
The input is 'consumed'.

This happens because you technically can't target a No Target cast so it is simply ignored.

Here's an example.

I have these items:
Glimmer Cape,
Gale Guard,
Eul's Scepter of Divinity
Magic Stick

To self-cast all these items I have to juggle the ALT key in an awkward way:

To Glimmer Cape myself, I have to use ALT
To Gale Guard myself (it can't be targeted), I have to release ALT
To Eul's Scepter of Divinity myself, I have to press ALT again
To use Magic Stick (it can't be targeted) I have to release ALT

I don't see the reason why it should be this way.
I'm sure at some point it'll get changed.
