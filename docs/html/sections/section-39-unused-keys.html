Unused keys

We've exhausted our left side of the keyboard in the most logical and practical way.
Which is exactly what we want.

What's left?
Not much actually.

N is far from Select # and Ability #. Easy to confuse with B
H same
Y requires hand movement
5 awkward for anything but Control Group 5
6 same, breaks hard position, hard to find
Win bad for many reasons, avoid

So... nothing, really?

Alt + Space weird, but available
Alt + F3 hard to use
Alt + F4 very hard to use

If anything - that confirms we've done a good job.

We might want to rearrange a key or two, but at this point this layout is extremely tight and moving one thing is going to affect - or even break - something else.

Also keep in mind that a mouse with 2 side buttons gives you not 2 - but 4 binds:

Mouse 4
Mouse 5
Alt + Mouse 4
Alt + Mouse 5

Which you might end up using eventually for something.

Random note

` ("grave") completely ignores Alt and Ctrl
It's possible to bind Alt+` but the game does not react to it.
