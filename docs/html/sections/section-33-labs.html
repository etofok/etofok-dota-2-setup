Labs

These are some next-level gameplay options.

Interact only with Allied Units can be a lifesaver in a clusterfuck of a fight to use Glimmer Cape, Decrepify, False Promise, etc.

Interact only with Hero Units might prevent you from wasting Doom on a creep.

Interact only with Non-Hero Units is a cheat code for snatching Bounty Runes or, previously, Wisdom Runes.

Another thing you might snatch is the Aegis.

Another great use for Interact only with Non-Hero Units is when you lane against heroes that put their bodies right on top of creeps.
You might accidentally attack their hero model instead, missing the last hit.

This is noticable against bulky heroes like Primal Beast or phasing heroes like Broodmother.

The problem is finding buttons for all these modifiers.

At the end of the day, you'd have to use these actions either with your pinky, thumb or - I don't know - maybe a pedal.

There are certain keyboards with the so called "macro buttons" to the left side of Tab, which I think might work well.
However since I don't have experience with that - I can't say for sure.

And if I had to choose only one - I believe Interact only with Non-Hero Units is the most impactful and easiest to use.
