Auto attack

Use Auto Attack: Never for Tinker and Earth Spirit
This gives you 100% intentional control over your hero.

But realistically Auto Attack: Standard is better for most use cases.

You probably want to have Summoned Unit Auto Attack: Always for manta and illusion based heroes such as Phantom Lancer, Terrorblade and Naga Siren.

This works: Auto Attack: Standard + Summoned Unit Auto Attack: Always

Another option is to use Toggle Autoattack button.

This allow you to use Summoned Unit Auto Attack: Same as Hero and switch the state with the button.

Don't forget to enable Quick Attack and Quick Move.
