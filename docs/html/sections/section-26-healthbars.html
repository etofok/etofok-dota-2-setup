Healthbar Notches

See these vertical 'notches' on the healthbar? Every notch represents 250 health points.

There used to be a way to change how 'big' a notch was.
But seems like it no longer works.

  • dota_health_per_vertical_marker 250

However, if you don't like the notches (for some reason) you can remove them.

  • dota_hud_healthbars 0 — no healthbars (for making videos)
  • dota_hud_healthbars 1 — simple healthbars for everyone
  • dota_hud_healthbars 2 — simple healthbars for enemies only
  • dota_hud_healthbars 3 — notched healthbars for everyone

Hurt indicator

When a hero is hurt you can see the white flash that indicates how much damage it took.

And we can modify it.

Below are the default values:

  • dota_health_hurt_decay_time_max 0.8
  • dota_health_hurt_decay_time_min 0.3
  • dota_health_hurt_delay 0.1
  • dota_health_hurt_threshold 0

The interesting one is the threshold one.
If you set the threshold value high — you won't ever see the white flash.
