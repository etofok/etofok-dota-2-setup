Select Units

There is another way to select and control units in Dota 2.

Dota 2 has these fantastic features:
Select Hero
Select All Other Units
Select All Units
Unified Orders

1.Select Herois self-explanatory.
The good thing is — you don't need it.

Just assign your hero to 1Control Group 1.
This gives you the same result, but provides additional flexibility of a control group.

Example:
Naga Sirenplayers usually run the main hero + one illusion together.

2.Select All Other Unitsselects all units under your control except your hero.

3.Select All Unitsselects all units under your control.
In other words,Select Hero+Select All Other Units.

4.Unified Ordersis a modifier key —Ctrlby default.
Unified Ordersallows us to command all units at once without selecting them.

This is very convenient when used occasionally.

These options should supplement your control groups — not replace them completely.

In my opinion there are 3 ways to approach these keybinds:

Option A - Bind these as ALT Control Groups
Option B - Bind these in place of some Control Groups
Option C - Bind these to some other keys

There are advantages and disadvantages to each.

Let's consider Option A:

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units:Alt  +   1
  • Select All Units:Alt  +   2

This makes sense.

But the problem is that we already use these buttons for our Shop actions.

The second problem isAlt+ key might be not very fast to use as our 'Select' keys anyway, so why bother.

Let's consider Option B:

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units: 2
  • Select All Units: 3

The main advantage here is that it's the fastest way to use these actions.

The downside is that you lose access to the control groups 2 and 3.
So this is worse when you actually want to control individual units or a specific group of units.

Another downside is that you lose continuity with other games in which 123 are control groups.

To be fair, you can still bind individual units to 4 5 and beyond.
Considering that you get instant access toSelect All Other UnitsandSelect All Unitsit might be a worthy trade-off.

Let's consider Option C:

If you're willing to sacrifice F1Select Courierand `Upgrade Talent...

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units: `   or   F1
  • Select All Units: F1   or   `

This might work because ` and F1 are close to 1, so all your "Select" keys are super close to each other.

You'd have to find keys forSelect CourierandUpgrade Talent, so probably not worth it.

Option D

  • Select All Other Units: WheelDown
  • Select All Units: WheelDown   +  Alt

Now, these are weird binds.
But this is what I'm currently using.

I manage most of my unit control using Control Groups anyway.

Fun Fact!

Naga Sirenhas illusions, but Terrorbladehas "images".

The difference?

You cannot assign individual images to different control groups.
When you assign an image to a control group that control group will always recall all images.

Not sure if intended.
